# 🛒 Amazon Clone - Frontend 

A fully responsive UI clone of the Amazon landing page built to practice CSS layouts and styling.

### 🎨 Features:
- **Navigation Bar:** Functional search bar and account menus.
- **Hero Section:** Smooth image sliders.
- **Product Grids:** Responsive cards for various categories.
- **Footer:** Exact replica of the multi-level footer.

### 🛠️ Tech Stack:
- **HTML5**
- **CSS3** (Flexbox & Grid)
